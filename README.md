# chan-dl.py
4Chan thread image archival script made in python

```
usage: chan-dl [-h] [-a URL] [-d URL] [-v] [-s] [-p Directory]

4chan thread image downloader.

optional arguments:
  -h, --help            show this help message and exit
  -a URL, --add URL     Adds url to thread list.
  -d URL, --delete URL  Removes url from thread list.
  -v, --version         Prints version info.
  -s, --start           Starts chan-dl on all the urls contained within the
                        thread list.
  -p Directory, --prefix Directory
                        Prefix for the download directory.

```
